# pagamentos.dev

pagamentos.dev is a unified SDK for payments in Brazil.

## Commands

- `bun run check`: Check lint issues.
- `bun run fix`: Fix (auto-fixable) lint issues.
- `bun run ./clone-repos`: Clone repos for `.context` folder

## .context

There's a folder in the root of the project named `.context`. This folder has
the source code of some libraries that might help with the development of the
current task.
